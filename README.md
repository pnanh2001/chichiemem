# chichiemem
Website chuyên về bán bánh ngọt, dụng cụ, hướng dẫn làm bánh và tất cả những thứ cẩn thiết cho người yêu bánh ngọt
Ngôn ngữ sử dụng: HTML, CSS, JavaScript và Python
Back-end sử dụng Django
